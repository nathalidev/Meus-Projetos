HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas da web.

Suas principais versões são:

Ano Versão
1991 1
1995 2
1997 3.2
1999 4.01
2014 5

Histórico do HTML:

  1. HTML: A primeira versão, lançada em 1991.
  2. HTML 2.0: Lançada em 1995, com melhorias significativas.
  3. HTML 3.2: Introduzida em 1997, com mais recursos como tabelas e formulários.
  4. HTML 4.01: Lançada em 1999, com aprimoramentos na estrutura e na formatação.
  5. XHTML: Uma versão reformulada do HTML, mais rígida com relação à sintaxe XML.
  6. HTML5: A versão mais recente e poderosa, lançada em 2014, com suporte a multimídia e interatividade avançada.

Tags HTML importantes:

  • <html>: Define o início e o fim do documento HTML.
  • <head>: Contém informações sobre o documento, como título e metadados.
  • <title>: Define o título da página exibido na barra de título do navegador.
  • <body>: Contém o conteúdo principal da página.
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Define os títulos de diferentes níveis.
  • <p>: Define um parágrafo de texto.
  • <a>: Define um link para outra página.
  • <img>: Insere uma imagem na página.
  • <ul>, <ol>, <li>: Cria listas não ordenadas (com marcadores), ordenadas (numéricas) e itens de lista, respectivamente.
  • <div>: Define uma divisão genérica ou contêiner.
  • <span>: Define uma área de texto em linha.
  • <table>, <tr>, <td>, <th>: Cria tabelas e suas respectivas linhas e células.
  • <form>, <input>, <button>: Cria formulários interativos para enviar dados.
  • <header>, <footer>, <nav>, <section>, <article>: Tags semânticas para melhorar a estrutura e acessibilidade do conteúdo.
  • <br>: Insere uma quebra de linha.
  • <hr>: Insere uma linha horizontal.
